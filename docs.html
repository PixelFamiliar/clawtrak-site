API Reference

Scan API

Free, public, rate-limited. No API key required.

Check any website's AI readiness score programmatically. The API runs the same 8-check scanner used by clawtrak.com and returns structured JSON with individual check results, a total score, and a letter grade.

Rate limit: 10 requests per minute per IP. Results are cached for 24 hours.

Endpoint

GET https://api.clawtrak.com/scan
ParameterTypeRequiredDescription
urlstringYesDomain or URL to scan (e.g. example.com)

Quick Start

cURL

curl "https://api.clawtrak.com/scan?url=shopify.com"

JavaScript

const res = await fetch('https://api.clawtrak.com/scan?url=shopify.com');
const data = await res.json();
console.log(data.score, data.grade);  // 37, "D"

Python

import requests
r = requests.get('https://api.clawtrak.com/scan', params={'url': 'shopify.com'})
data = r.json()
print(f"{data['score']}/100 — Grade {data['grade']}")

Response

A successful scan returns a JSON object with the following fields:

{
  "url": "shopify.com",
  "reachable": true,
  "score": 37,
  "max_score": 100,
  "grade": "D",
  "checks": [
    {
      "name": "llms.txt",
      "score": 15,
      "max": 15,
      "status": "PASS",
      "detail": "/llms.txt found and well-formatted (21 lines)."
    },
    {
      "name": "AGENTS.md",
      "score": 0,
      "max": 10,
      "status": "FAIL",
      "detail": "No AGENTS.md found."
    }
    // ... 6 more checks
  ],
  "summary": "Score: 37/100 (Grade D)",
  "timestamp": "2026-03-14T21:30:00"
}

The 8 Checks

CheckMax PointsWhat It Tests
llms.txt15AI profile file at /llms.txt
AGENTS.md10Agent rules of engagement
Agent Card15Discovery card at /.well-known/agent
Bot Permissions10AI bot directives in robots.txt
Structured Data15JSON-LD embedded in HTML
Business Schema10Schema.org vocabulary coverage
API Spec15OpenAPI/Swagger specification
Page Metadata10Meta tags, headings, semantic HTML

Grading Scale

  • A (80-100) — Fully agent-ready
  • B (60-79) — Mostly ready, minor gaps
  • C (40-59) — Partially visible to AI
  • D (20-39) — Barely visible
  • F (0-19) — Invisible to agents

Error Responses

// Missing URL parameter
{"error": "url parameter required"}           // 400

// Rate limit exceeded
{"error": "rate limit exceeded, try again in 60s"}  // 429

// Site unreachable
{"error": "Could not reach example.com"}      // 500
